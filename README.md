# 📝 Todo-лист с фильтрацией

Простой и красивый список задач с возможностью:
- ✅ Добавлять задачи
- ✅ Отмечать выполненные
- ✅ Удалять задачи
- ✅ Фильтровать: Все / Активные / Выполненные
- ✅ Данные сохраняются в браузере (localStorage)

## Как запустить
Просто открой `index.html` в браузере!

## Технологии
- HTML
- CSS
- JavaScript (чистый, без библиотек)
- localStorage для хранения данных
